## 如何新增 ADR

1. 复制 `template.md` 为 `NNNN-short-title.md`。
2. 使用递增编号。
3. 填写所有必要章节。
4. 在同一个 PR 中更新本索引。

## 状态说明

- **提议中**：正在讨论，尚未达成共识
- **已接受**：已经采纳
- **已实施**：实现已经落地
- **已废弃**：不再推荐
- **已取代**：被另一个 ADR 替代
